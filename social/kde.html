Lemon's Resting Area

KDE

KDE は、デスクトップ環境を中心に、様々なデスクトップアプリケーションと、それらに必要なコンポーネントの開発を行う、国際的なオープンソースプロジェクトです。

—Japan KDE User Group, What is KDE

KDE での貢献は、 旧 Lemon Electic Computer 世代から NEO で行っていました。

茶屋計画後の Lemon's Resting Area でも継続して KDE の様々な活動に参加しています。

私が行っている活動は以下の通りです。

  • 翻訳
  • 開発
  • 広報
  • 資料
  • デバッグ

翻訳

翻訳活動は2022年11月から開始し、その後も不定期に活動しています。

現在までに翻訳を行った主なプロジェクトは以下です。

  • ソフトウェア系
    • Kontrast (100%)
    • Filelight (60%)
    • Krita
    • Kdenlive
  • ウェブサイト系
    • KDE 本部
    • Krita
    • Kdenlive Docs
    • KDE Userbase Wiki

開発

ウェブサイトの微調整や開発資料の修正などの微細な点で貢献しました。

  • アーカイブのアイコンがつぶれる問題を修正 (KDE Japan Website !14) [KDE Invent]
  • Responsive design (KDE Italy Website !1) [KDE Invent]
  • README: update krita ci badge (Krita !2037) [KDE Invent]
  • (依頼) Request: Redirect from https://krita.org/jp/* to https://krita.org/ja/* (Krita Website 98094) [Krita Artists]
  • Remove the old link (Krita.org !64) [KDE Invent]

KDE で主に利用されている言語は C++ や Python で、私が得意としている C# はほとんど利用されていないため、ソフトウェア関連の開発貢献はあまりできていません。

広報

日本 KDE ユーザー会の一員として、日本国内で KDE を普及させることを目的とした活動にも参加したいと考えています。

イベント開催の主催も行いたいと思っていますが、目処は立っていません。

Lemon's Resting Area

プロフィール

Lemon73 / S. Hagoromo

こんにちは。 Lemon's Resting Area の Lemon73 と申します。

コーディング、デザイン、動画編集など幅広い活動を行っています。

コーディング

  • HTML + CSS
  • C#
    .NET 信者。
  • JavaScript / TypeScript
    他の名義での開発の際に React.js と React.ts を使ったことがある。
    Vue.js もちょっとだけ。

ツール

  • OS
    • KDE Neon
      Ubuntu ベースで、デスクトップ環境として KDE Plasma を採用した Linux 系 OS 。
      KDE の KDE による KDE のためのディストリビューション。
    • Windows
      今もたまに使ってる。
  • デザイン
    • Krita
      世界最高峰のイラストソフトウェア。
      一部翻訳してる。
    • Medibang Paint
      Krita を見つけるまではメインだった。
    • GIMP
      画像編集あんまりしないので、ほとんど使ってない。
      簡単な画像編集なら Krita の G'mic Qt 使えばいいし…
    • Inkscape
      アイコン作るならこれ。
      KDE の Karbon は開発停止になったので、 Linux では実質これしか選択肢がない。
  • 3DCG
    • Blender
      最高のモデリングツール。
    • MMD
      Windows 時代は使ってた。
  • ゲーム開発
    • Unity
      昔に諦めた。 Godot に挑戦してみたい。
      ちなみに、 UE はパソコンのスペック的にそもそも動かなかった。
    • RPG ツクール VX Ace / RPG Maker MV
      LRA (当時の LEC) 設立以前の数年間使ってた。
    • Alice Novel
      チョットワカル。
  • コーディング
    • VSCode
      Visual Studio は Linux 対応していないから、 C# を書く際も結構使っている。
      Visual Studio が重いから、 Windows でも Blazor の開発の際は VSCode + コマンドが多い。
      最近 VSCode (on Linux) で MAUI Android にビルドする方法がわかったので、より利用頻度が増えた。
    • Visual Studio
      Windows 向けの .NET MAUI のビルドするときとかに使う。
    • NetBeans IDE
      昔 Java の SkyColors を作ったときに使った。
      Java はよくわからない。
    • Arduino IDE
      別の名義で使った。
      よくわからない。
    • Qt Creator + Qt Design Studio
      Qt は諦めたけど、もう一度挑戦してみたい。
      QML 書けるようになりたいなあ。
    • Git
      義務教育。
      とか言いながら、コマンドはほぼ使ってない。
    • GitHub Desktop
      昔はこれを使ってた。
      今は VSCode の Git 操作ツールを使ってる。
    • FileZilla
      ファイル送信プロトコル。
      日本人として FFFTP を使えよ。
      ウェブサイトが GitLab Pages (今は GitHub Pages) に移管したことで使わなくなった。
    • Konsole + .NET CLI
      C# の開発に欠かせないセット。
  • SNS
    公開しているものはこちら。
    以下に記載するのは、個人で利用している (非公開の) SNS 。
    • LINE
      あんまり好きじゃない…
    • Skype
      昔は使ってたけど、今はもう…
    • Slack
      割と無料枠が狭い。
      課金して使うならいいサービスだと思う。
      (まあ、会社での利用じゃなかったら Discord とか Matrix でいい気がするけど)
  • 音楽
    • LMMS
      最近音楽作ってない。
    • Domino
      Midi 系の作曲ツール。
      Windows 限定だから Linux では使えない。
    • Synthesizer V Studio Basic
      最新の歌える人工音声。すごい。
  • 音声
    • ゆかりねっと
      昔、生配信をやろうとしていたときに使おうと思っていた。
    • Voice Vox
      定番の。動画使ったことないけど。
    • Coeiroink
      自分が持っているのは、 Voice Vox と UI が共通だった世代のもの。
      (今は独自 UI になっている)
    • SofTalk
      自分が持っているのは、 AquesTalk が使える時代のもの。
  • 動画編集
    • Kdenlive
      日本語情報が少なすぎて使いやすくはない。
      一部翻訳してる。
    • AviUtl
      OSS にしてくれ… (懇願)
    • YMM3 / YMM4
      実は YMM2 を使ったことがある世代。
      逆に、最新の YMM4 はよくわからない。
  • その他
    • Lokalize
      KDE のソフトウェアを翻訳するためのソフトウェア。
    • Kando
      v0.1.0 からのヘビーユーザー。
    • KDE Connect
      パソコンと携帯端末の連携が強くなる。
      具体的には、クリップボードの共有や、ファイルの送信、スマホをマウス代わりにしたり、プレゼン資料や音楽の操作などあらゆることができる。
    • Opera / Opera GX
      パソコンを始めたときからのお供。
      実は、現代のモダンブラウザの中で随一の歴史を誇る伝統的なブラウザ。
      近年も AI の Aria を搭載するブラウザとして、進化を続けている。
      なお、ゲーム向けの Opera GX は Linux に対応していない。
